Web Resources

Here is some info to learn HTML and CSS

  1. Mozilla: Mozilla's developers guide to HTML. Similar to W3Schools, you can look up html code here
  2. HTML5up : Free HTML and CSS template sites. This is good for getting ideas and seeing example of code
  3. CSS-tricks: CSS-Trick's cheatsheet for flexbox. This is a good, basic overview of flexbox commands
  4. Meyerweb: CSS reset to ensure browser conformity with CSS presentation
  5. Codepen: Simple developer sandbox for testing out ideas. Codepens can also be saved and posted where you want them
